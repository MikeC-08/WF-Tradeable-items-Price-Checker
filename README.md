# Warframe Market 價格查詢

## 使用方法
初始化：選擇隊伍人數(可隨時更改)，並按下開始。
1. 等待選擇遺物階段，按下\<Enter\>鍵，開始查詢價格。(可能因爲Warframe Market繁忙而卡住，届時需要重開)
2. 待完結後，再次按下\<Enter\>鍵關閉界面。
